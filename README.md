# Convertidor de Unidades

Este es un proyecto web simple desarrollado en Python usando Flask. Permite convertir unidades de longitud, peso y temperatura, realizando los cálculos del lado del servidor.

## Requisitos previos

- Python 3.x instalado en tu sistema.
- pip (gestor de paquetes de Python).

## Instrucciones de ejecución

1. Clona o descarga este repositorio.
2. Abre una terminal y navega hasta la carpeta del proyecto.
3. Instala la dependencia necesaria (Flask) ejecutando el siguiente comando:
   ```bash
   pip install Flask
